## ğŸš€ About ShiftPlanConverter

ShiftPlanConverter is a web application that converts shift schedule PDFs and synchronizes them with Google Calendar. It processes all data locally in the browser for maximum privacy and security.

### Key Features
- **Local PDF Processing**: No data uploads, everything happens in your browser
- **Automatic Shift Recognition**: Detects shifts, vacation, and holidays automatically
- **Google Calendar Integration**: Direct synchronization with your calendar
- **Customizable Shift Types**: Support for various hospital departments and shift patterns
- **Multiple Export Formats**: CSV download and ICS export options

### Technology Stack
- **Frontend**: Vanilla JavaScript, HTML5, CSS3 (Tailwind CSS)
- **PDF Processing**: PDF.js library
- **Calendar Integration**: Google Calendar API
- **Deployment**: Docker with Nginx
- **Security**: OAuth 2.0, local data processing
